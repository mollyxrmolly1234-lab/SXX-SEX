Connect to WhatsApp

Quick Setup Guide:

  1. Enter your WhatsApp number with country code (no "+", no brackets, no spaces)
  2. Click "Request Pairing Code"
  3. Copy the 8-digit code provided
  4. Open WhatsApp → Settings → Linked Devices → Link a Device
  5. Enter the code when prompted
